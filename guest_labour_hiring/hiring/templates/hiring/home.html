01 · About

The System
India Needs

Millions of migrant workers move across India every year without proper documentation or employer verification. This platform bridges that gap.

Built as an IGNOU BCA Final Year Project, it uses Django with role-based authentication ensuring each stakeholder only sees what they need.

Key
Features

Every module built with precision — NOC issuance, job management, worker tracking and more.

01
Police Verification

Workers are verified by police who issue or reject No Objection Certificates before employment.

02
Job Management

Employers post vacancies. Workers apply and track applications in real-time.

03
Worker Registry

Complete migrant database with Aadhaar, DOB, languages, and employment history.

04
Role-Based Security

4 separate dashboards. Only Admin creates accounts. CSRF protection throughout.

05
Insurance Tracking

Admin manages worker insurance — type, coverage name, and payout amounts.

06
MySQL / SQLite DB

Works with SQLite out of the box. One config line switches to MySQL Workbench.
